
# dbMagic
dbMagic is a simple python script which allows you to edit .sql file without the need of MySQL th allows you to just edit

### Prerequisites

    Installations
        - python3
        - tqdm

### Run
	$ python3 dbMagic.py
	usage: dbMagic.py [-h] -s SQL [-c CHANGELIST] [-t TOCHANGELIST] -m MATCH [-e]
